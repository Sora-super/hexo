给定两个数组，编写一个函数来计算它们的交集。

示例1:

```
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2,2]
```

示例2:

```
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[4,9]

```

看到题目直接采用存哈希表解决


```js
function intersect(nums1, nums2){
    const map = {}
    const res = []
    for (let x of nums1){
        if (map[x]){
            map[x]++
        } else {
            map[x] = 1
        }
    }

    for (let y of nums2) {
        if (map[y] > 0) {
            res.push(y)
            map[y]--
        }
    }
    return res
}
```
