## 安装

根据文档可傻瓜式安装

``` bash

1. npm install -g hexo-cli  # 全局安装hexocli

2. hexo init <projectname>  # 找一个空闲的目录，创建博客仓库

3. cd  <projectname>  # 进入目录

4. npm i        # 安装依赖

5. npm run server # 执行开发命令 默认端口4000

6， npm run build # 打包文件，生成public静态目录，直接把public丢到服务器上即可
```
